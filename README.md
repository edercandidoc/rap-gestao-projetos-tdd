# Gestão de Projetos - SAP RAP & TDD 🚀

Projeto desenvolvido em **ABAP Cloud** (SAP BTP Trial) utilizando o framework **RAP (RESTful ABAP Programming Model)**.

## 🛠️ Tecnologias Utilizadas
- **ABAP 7.5x / ABAP Cloud**
- **SAP RAP (Managed Scenario)**
- **OData V2**
- **CDS Views & UI Annotations** (Criticality, Icons, Value Help)
- **TDD (ABAP Unit)** com CDS Test Double Framework

## 📋 Funcionalidades
- CRUD completo de projetos.
- Cálculo automático de duração de dias via HANA (campo calculado).
- Validação de datas (Data Fim > Data Início).
- Determinação automática de status inicial.
- Interface rica com cores e ícones baseados no status.

## 🧪 Testes Unitários
O projeto conta com cobertura de testes unitários para as validações de negócio, garantindo a integridade dos dados e facilitando a manutenção futura.
